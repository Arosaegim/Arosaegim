## Getting Start

0. 한번이라도 saegim.me에 접속한 적이 있다면 크롬설정에서 쿠키를 지우자!
1. 구글 크롬으로 saegim.me에 접속한다.
2. 자동으로 '홈 화면에 추가'가 팝업으로 뜨면 추가해주고, 안뜨면 설정에 들어가서 수동으로 홈 화면을 추가.
